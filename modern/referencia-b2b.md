# Diretrizes da versão moderna SKW (B2B)

## Correção para erro de PR com binários

- Removidos assets binários versionados (`.png/.jpg`) do diretório `modern/` para evitar erro de compatibilidade na criação do PR via Codex.
- Portfólio agora usa **thumbnails visuais em CSS** (gradientes e padrões), mantendo a identidade preto/branco com amarelo `#febc29`.
- A marca no header foi ajustada para versão tipográfica (`SKW VIDEO`) sem dependência de arquivo de imagem.

## Referências usadas (direção de portfólio B2B)

- Demo Duck: destaque visual no topo do portfólio.
- Wyzowl: organização simples por tipo de entrega.
- Sandwich Video: leitura rápida de cada case.

## Próximo passo recomendado

Quando o ambiente de PR aceitar binários, reintroduzir capas reais em formato otimizado (WebP/AVIF) e manter fallback CSS.
